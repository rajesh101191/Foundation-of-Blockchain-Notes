# Foundations of Blockchain Technology

Academic teaching resources developed for the undergraduate course **BT501 – Foundations of Blockchain Technology**.

## Topics Covered

* Blockchain Fundamentals
* Distributed Ledger Technology (DLT)
* Cryptographic Hash Functions
* Digital Signatures
* Merkle Trees
* Consensus Mechanisms
* Proof of Work (PoW)
* Proof of Stake (PoS)
* Ethereum and Blockchain 2.0
* Smart Contracts
* Solidity Basics
* Blockchain Governance
